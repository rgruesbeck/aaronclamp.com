# aaronclamp.com

Website aaronclamp.com
```sh
npm install
bower install
```

## Get Started
Start gulp build
```sh
npm start
```
Open a new shell and start the server
```sh
npm run server
```

## Deploy
```sh
npm run build
npm run deploy-staging/deploy-production
```
